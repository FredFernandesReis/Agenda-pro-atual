# 📢 Como Usar o Sistema de Anúncios

## 🎯 O que é?

Sistema de anúncios que permite você comunicar atualizações, avisos e informações importantes para todos os usuários do sistema.

## ✨ Como Funciona?

1. **Super Admin cria um anúncio**
2. **Anúncio aparece no topo da tela** para todos os usuários logados
3. **Usuário pode fechar** clicando no X
4. **Uma vez fechado, não aparece mais** para aquele usuário

## 📝 Como Criar um Anúncio

### Passo 1: Acessar Gerenciamento de Anúncios

1. Faça login como **Super Admin**
2. No menu lateral, clique em **"Anúncios"**
3. Clique em **"Novo Anúncio"**

### Passo 2: Preencher Dados

- **Título**: Título do anúncio (ex: "Nova Atualização!")
- **Mensagem**: Texto completo do anúncio
- **Tipo**: Escolha a cor:
  - **Informação** (Azul) - Para atualizações gerais
  - **Sucesso** (Verde) - Para boas notícias
  - **Aviso** (Amarelo) - Para alertas
  - **Importante** (Vermelho) - Para mensagens críticas
- **Data de Expiração**: Opcional - quando o anúncio deve parar de aparecer
- **Ativo**: Marque para o anúncio aparecer

### Passo 3: Salvar

Clique em **"Salvar"** e pronto! O anúncio já aparece para todos os usuários.

## 🔄 Como Editar um Anúncio

1. Vá em **"Anúncios"**
2. Clique no ícone de **lápis** do anúncio
3. Edite os dados
4. Salve

## 🗑️ Como Excluir um Anúncio

1. Vá em **"Anúncios"**
2. Clique no ícone de **lixeira** do anúncio
3. Confirme a exclusão

## 💡 Dicas de Uso

### Exemplos de Anúncios:

**Atualização do Sistema:**
- Tipo: Informação
- Título: "Nova Versão Disponível"
- Mensagem: "O sistema foi atualizado com novas funcionalidades. Confira!"

**Manutenção Programada:**
- Tipo: Aviso
- Título: "Manutenção Programada"
- Mensagem: "O sistema ficará indisponível no dia X para manutenção."

**Nova Funcionalidade:**
- Tipo: Sucesso
- Título: "Nova Funcionalidade!"
- Mensagem: "Agora você pode fazer X diretamente do dashboard!"

**Aviso Importante:**
- Tipo: Importante
- Título: "Atenção!"
- Mensagem: "Lembre-se de renovar sua assinatura até dia X."

## ⚙️ Funcionalidades Técnicas

- ✅ Anúncios aparecem apenas para usuários logados
- ✅ Cada usuário pode fechar o anúncio
- ✅ Uma vez fechado, não aparece mais (salvo no navegador)
- ✅ Você pode definir data de expiração
- ✅ Pode ativar/desativar a qualquer momento
- ✅ Múltiplos anúncios podem estar ativos ao mesmo tempo

## 🎨 Tipos de Anúncios

- **Informação (Azul)**: Use para comunicar atualizações, novidades gerais
- **Sucesso (Verde)**: Use para boas notícias, conquistas
- **Aviso (Amarelo)**: Use para alertas, lembretes importantes
- **Importante (Vermelho)**: Use para mensagens críticas, urgências

## 📱 Onde Aparece?

Os anúncios aparecem no **topo de todas as páginas** do sistema, logo após o login, antes do conteúdo principal.
